# Synthetic Social Evidence Review Context

This synthetic context file is a placeholder target for a caller-owned social evidence review. It contains no real account names, customer data, campaign details, publication plan, private repository paths, or brand strategy.

## Intended Surface

A public page or content draft wants to cite community response, testimonials, or usage signals without overstating what those signals prove. The reviewer should treat this file as a neutral evidence packet, not as a source of truth about a real project.

## Review Boundary

Flag unclear claims, missing caveats, unsupported generalizations, audience mismatch, and places where sanitized social proof might imply stronger evidence than the packet provides. Do not infer private project facts, approve publication, set brand strategy, or route follow-up work.
